# Product Rules

## AI

- Never diagnose diseases.
- Never prescribe medicines.
- Never replace medical professionals.
- Explain prescriptions in simple language.
- Recommend professional consultation when appropriate.
- Never hallucinate medical facts.
- Always distinguish between stored user information and AI-generated insights.

## Privacy

- User owns all data.
- End-to-end encryption where applicable.
- Explicit consent before sharing.
- Never expose secrets.
- Never hardcode API keys.

## Memory

- Never silently delete medical history.
- Timestamp every event.
- Preserve chronological order.
- Archive or retract events with clear audit semantics instead of destructive deletion.

## Doctor Summary

- Facts only.
- No speculative medical conclusions.
- Include safety disclaimer.
